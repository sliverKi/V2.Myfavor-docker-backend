> 주요기능

- 인증&로그인
1. 회원가입시 client 이메일 인증
2. 회원가입시 기입한 정보의 유효성 검사
3. 아이디 찾기
4. 비밀번호 찾기시, client mail주소로 비밀번호 초기화 하는 링크 전송
5. 비밀번호 변경

- 아이돌 조회
1. 그룹, 솔로 이원화
2. 아이돌 그룹의 Detail 정보 조회 가능
   -  멤버, 앨범, 스케쥴 정보 조회 가능
3. 솔로 아이돌의 Detail 정보 조회 가능
   - 앨범, 스케쥴 정보 조회 가능
